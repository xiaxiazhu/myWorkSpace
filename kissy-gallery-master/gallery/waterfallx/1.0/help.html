作者：tafeng.dxx@taobao.com
功能：踏风
描述：重写的瀑布流
源码：waterfallx.js

What's this?

就是个瀑布流，与kissy最大的不同是，采用了inline-block布局，而未采用KISSY的绝对定位布局

如何用？

与KISSY的瀑布流使用方式一致，接口兼容KISSY的瀑布流

优点

1.容器无需定高；
2.动态改变容器高度时，无需调用adjust，因为容器是静态定位的：）
3.adjust的平均效率会更高
4.支持在瀑布流头部插入数据，这可以通过调用preAddItems
5.API兼容KISSY，并多了一个preAddItems用于前向插入数据

适用场景

图片尺寸未知且需要自适应, 线上案例：

淘宝周边

有图有真相

其他

  • 1.弹性布局(如width:70%)，若出现折行现象，请设置瀑布流容器的white-space:nowrap; 主要原因是该瀑布流未采用绝对定位， 通过margin来控制布局，会导致百分数的情况下，四舍五入多出1像素的margin，从而导致折行现象！
  • 2.preAddItems的参数与addItems一致！

API

参考：KISSY WaterFall

新增加的方法：preAddItems：从顶部插入数据，与addItems的参数一致。
