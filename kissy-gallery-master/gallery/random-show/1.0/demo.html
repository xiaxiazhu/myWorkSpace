API

CLASS

RandomRegion(config) - 在固定区域取若干随机位置、大小区域

  • 参数

    • config: {Object} –配置项
      • width: {Number} –区域宽度
      • height: {Number} –区域高度度
      • sizes: {Object|Array} – 单个区域大小

        Object:{baseWidth:(Number),baseHeight:(Number),minRatio:((Float))} - 设置基本宽高和最小缩小比例

        Array:[{width:Number,height:(Number)},{},...] - 直接设置各个区域的大小

方法

getRegions ( count ) - 获取区域

  • 参数

    • count:{Number} - 要取区域的数量

  • 返回

    {Array}:[regionObj,regionObj,...] - 各个区域对象的数组

    regionObj:

    • top: {Number} - 区域上边界位置
    • right:{Number} - 区域右边界位置
    • bottom:{Number} - 区域下边界位置
    • left:{Number} - 区域左边界位置

CLASS

RandomShow(container[, config]) - 基于RandomRegion,UI层的效果封装

  • 参数

    • container: {String|HTMLElement} – 容器
    • config: {Object} –可选，配置项
      • width: {Number} –区域宽度，默认为容器宽度
      • height: {Number} –区域高度度，默认为容器高度
      • sizes: {Object|Array} – 单个区域大小，默认为首个子元素大小,最小缩放比例为0.6

        Object:{baseWidth:(Number),baseHeight:(Number),minRatio:((Float))} - 设置基本宽高和最小缩小比例

        Array:[{width:Number,height:(Number)},{},...] - 直接设置各个区域的大小

方法

run ( ) - 执行初始动画
stop ( ) - 停止初始动画

Note

  • 随机展示，需要设置合适的单个区域大小，如果查找不到会被隐藏
