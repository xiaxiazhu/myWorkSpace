当前位置： KISSY Gallery Amount 
                
作者：bofang.zxj@taobao.com(伯方)
功能：amount
描述：数字金额转换成中文大写
源码：amount.js

Demo

显示源码

API

方法

            KISSY.use('gallery/amount/1.0/index', function(S, Amount) {
                var am = new Amount('1235.23',true);
                S.log(am.convertAmount());
            });

参数

  • num,需要转换的目标数字
  • hasDecimal,是否需要小数点，默认为false

返回

convertAmount 返回值为 string

其他

组件只是对数字进行转换，对传入不合法的参数，比如字母、标点、开头很多0，多个小数点等情况都做了容错处理，不涉及dom操作
