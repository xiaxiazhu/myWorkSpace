当前位置： KISSY Gallery Huabao API 
作者：法海 <fahai@taobao.com>
功能：huabao/ui/detail
描述：ui 装饰器，单例。提供画报详情页的 ui 风格，包括滚动的缩略图区还有大图区
源码：ui/detail.js

API

方法

static Slide | Carousel decorate(huabao, config)

参数

  • {Huabao} huabao 需要添加装饰的画报对象
  • {Object} config 配置对象
    • {String} target 装饰的目标。两个选项： "content" 画报大图区，内部实现使用 Slide 组件装饰； "nav" 缩略图导航区，内部实现使用 Carousel 组件装饰；
    • {Array<HTMLElement>} panels 对应 switchable 中的 panels，默认为 huabao.get("panels")
    • 其他配置项请参考 Slide 和 Carousel

返回

  • {Slide} slide 当 target 为 content 时则返回一个 Slide 实例
  • {Carousel} carousel 当 target 为 nav 时则返回一个 Carousel 实例
