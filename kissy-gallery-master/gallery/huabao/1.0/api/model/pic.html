当前位置： KISSY Gallery Huabao API 
作者：法海 <fahai@taobao.com>
功能：huabao/model/pic
描述：此类是对“画报中的图片”的建模，一个 Pic 实例代表一个画报图片。
　　　Pic 类主要定义了一些标准的属性字段，并且提供一个可被重写的 parse 方法。
　　　Pic 类从 Base 继承而来，使用标准的 get / set 方法来访问字段。
源码：model/pic.js

API

属性

  • {String} id 图片 id
  • {String} picSrc 图片地址
  • {Int} width 图片宽度
  • {Int} height 图片高度
  • {String} desc 描述
  • {Array} tags 标签列表
  • {Object} rawData 以 json 形式存放的所有字段，只读。 方便给 kissy template 调用……

方法

Object parse(dataSource)

参数

  • {Object} dataSource 数据源对象

返回

  • {Object} rawData 返回一个 json，其所有字段名和 Pic 的相应字段名一致，值类型也一致。 默认的 parse 方法原封不动地返回 dataSource 对象，因此如果数据源的字段名不符合要求，需要重写 parse 方法
