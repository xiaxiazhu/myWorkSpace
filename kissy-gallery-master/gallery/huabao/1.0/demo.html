当前位置： KISSY Gallery Huabao 
作者：法海 <fahai@taobao.com>
功能：Huabao
描述：淘宝画报的 model 抽象。通过注入自定义的业务逻辑和 ui 装饰，使用者可快速建立出一个画报成品。
　　　默认包含一套 ui 装饰，使用者可以自行定义。
源码：huabao.js

Demo

Demo 包含一个列表页和详情页。点击列表中的图进入其详情页。

构建一个画报应用，大体分为几个步骤：

  1. 获取数据源。数据源可以是：页面中的 json；通过 ajax 返回的远程数据。 数据结构需要参照 Huabao.model 中的 Poster 和 Pic； 如果数据源提供的数据结构和 model 的不一致，那么需要自己重写 Poster.prototype.parse 和 Pic.prototype.parse 方法
  2. 生成画报对象
  3. 使用预定义或自定义模板渲染
  4. 添加业务逻辑。业务逻辑一般在 attrChange 的时候触发
  5. 添加 ui 装饰，可使用已有的 ui widget 或自定义对画报进行装饰、绑定物理事件等

详情和源码请参考 demo (列表页, 详情页)
