当前位置： KISSY Gallery Sliding Labels 
作者：qiaohua@taobao.com(乔花)
功能：表单label浮动功能
描述：表单各个输入项目中, 如果有label的话, 可以很方便地放在输入框上方, 当点击时移开label;
源码：sliding-labels.js

Demo

显示源码

API

方法

new SlidingLabels( trigger, config )

参数

  • {String} [ trigger = '#info' ] 触发元素
  • {String} [ config = '{} ] 配置项, 见下:

配置项

  • {Array} [ position = [5, 5] ] px, 水平和垂直方向上, 相对于父元素的位置, x or [x, y], 不设置时, 取 [5, 5]
  • {Number} [ offset = 5 ] label 和 input 之间的偏移距离, 默认为 5
  • {Number} [ zIndex = 99 ] zIndex, 默认为 99
  • {Number} [ duration = 0.2 ] 动画速度, 默认为 0.2
  • {Object} [ focusStyle = {} ] 输入框获取焦点时, label 的样式, 默认为 undefined
  • {Object} [ blurStyle = {} ] 输入框失去焦点时, label 的样式, 默认为 undefined

返回

Object

Note

  • 目前只需要初始化, 没有其他方法支持.
